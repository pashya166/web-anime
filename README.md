# web-anime

`web-anime` adalah sebuah proyek aplikasi web frontend yang berfungsi sebagai **katalog, penelusur, dan penampil detail informasi anime**. Proyek ini bertujuan untuk menyediakan antarmuka yang sederhana dan cepat bagi pengguna untuk mencari serial dan film anime favorit mereka.

---

## âœ¨ Fitur Utama

* **Katalog Anime Dinamis:** Menampilkan daftar anime yang dimuat secara dinamis, seringkali melalui API publik (misalnya Jikan API atau sejenisnya).
* **Fungsi Pencarian Cepat:** Memungkinkan pengguna untuk mencari anime berdasarkan judul dengan *live filtering* atau *request* API.
* **Tampilan Detail Anime:** Menyajikan halaman terpisah untuk setiap anime, menampilkan sinopsis, genre, skor/rating, dan informasi staf.
* **Antarmuka Responsif:** Dirancang menggunakan CSS modern agar tampil optimal di berbagai ukuran layar (desktop, tablet, dan *mobile*).
